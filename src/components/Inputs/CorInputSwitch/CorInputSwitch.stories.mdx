# CorInputSwitch

A `CorInputSwitch` allows the user to switch two option.

## Props

| Prop  | Description | Type | Required? | Default Value |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| id | switch id (similar to HTML id) | string |  | |
| className | class to apply on component| string or string[] |  | |
| label | label shown next to input | string | | |
| checked | whether the switch is checked or not by default | boolean | | false |
| disabled | whether the input is disabled or not | boolean | | false |
| inline | set the position of switch and label in one line | boolean |  |false |
| labelPosition | set label position | 'left' | 'right' | | 'left' |

## Methods

| Prop | Description | Type | Required? | Default Value |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| onChange | called when value changes | (value: boolean) => void |  | |
