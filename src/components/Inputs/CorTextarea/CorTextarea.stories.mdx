# CorTextarea

A `CorTextarea` allows the user to enter a multiline text.

## Props

| Prop  | Description | Type | Required? | Default Value |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| maxLength | the maximum input length in input | number | | |
| minLength | the minimum input length in input | number | | |
| disabled | whether the input is disabled or not | boolean | | false |
| value | the value of the button can be binded to `v-model` | string | | |
| label | label shown above the input | string | | |
| statusMessage | a small message shown under the input | string | | |
| placeholder | a placeholder that will shown when the input is empty | string | | |
| status | input status | 'success', 'warning', 'error' | | |
| class | CSS classes | string | | |
