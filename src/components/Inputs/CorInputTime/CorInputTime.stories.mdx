# CorInputTime

A `CorInputTime` allows the user to specify time with 24 or 12 format.

## Props

| Prop  | Description | Type | Required? | Default Value |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| classes | custom css clases to use in this component id (similar to HTML id) | string or string[] |  | |
| maxTime | Date max to use | Date |  | |
| maxTime | Date max to use | Date |  | |
| timeFormat | type of format to render date 24 or 12 with second milliseconds | 'h', 'HH', 'h:mm', 'HH:mm', 'h:mm:ss', 'HH:mm:ss','h:mm:ss:S', 'HH:mm:ss:S' | | 'HH:mm' |
| disabled | whether the input is disabled or not | boolean | | false |
| value | default value render in the input| Date |  | now |
| customIcon | override the default checkbox SVG | VNode, VNode[] | | |
| checkPosition | set checkbox position | 'left' | 'right' | | 'left' |

## Methods

| Prop | Description | Type | Required? | Default Value |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| onChange | called when value changes | (event: Event, newTime: Date) => void | ✅ | |
| onBlur | called when focus out | (event: Event, newTime: Date) => void | ✅ | |
| onKeydown | called when press the arrow up or down key | (event: KeyboardEvent, newTime: Date) => void | ✅ | |
